Keyboard Shortcuts: Ctrl/Cmd+B Bold • Ctrl/Cmd+I Italic • Ctrl/Cmd+K Link • Ctrl/Cmd+` Code • Ctrl/Cmd+1-6 Headings
Indentation: Tab Indent • Shift+Tab Unindent
Links: [text](url) creates a hyperlink • Type [ to auto-pair brackets • Ctrl/Cmd+K wraps selection in link
Auto-Lists: Type * or - for bullets • 1. for numbers • - [ ] for checkboxes • > for quotes
Auto-Pairing: ** ` [ auto-complete • Press Enter in lists to continue
